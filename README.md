Little encryption tool in C
======

This program take a string as unique parameter and output this string with the folowing encryption :  
reverse -> caesar (key is 12) -> base64
  
example:
--------
     - input string : "salut"  
     -> reverse : "tulas"  
     -> caesar : "fgxme"  
     -> base64 : "Zmd4bWU="
